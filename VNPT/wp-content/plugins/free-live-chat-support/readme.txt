== Installation ==

Installation: https://vimeo.com/202022178

Step-by-step Guide

• Install the ZupportDesk plugin from the WordPress directory and activate it

• Click on the ZupportDesk icon on the left panel in WordPress and log in to your account

• Select the Chat Widget from the widget list and hit ‘Next’

• Log in to your Dashboard and start chatting!

- Do I have to install any software on my server? No, ZupportDesk is a 100% cloud-based service. Just configure the plugin and your Live Chat Widget is ready to go.
- Are different web browsers supported? Yes, ZupportDesk works on most browsers and it is optimized for Internet Explorer, Google Chrome, Mozilla Firefox, Safari and Microsoft Edge.
